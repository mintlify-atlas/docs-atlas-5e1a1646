# Welcome to Nuxt Lettermint

A powerful Nuxt module for sending transactional emails using the [Lettermint](https://lettermint.co) email service. This module provides seamless integration with Lettermint's Node.js SDK for both server-side and client-side email sending capabilities.

<Note>
  Lettermint is a European transactional email service provider focused on simplicity, reliability, and developer experience.
</Note>

## Why Lettermint?

Lettermint is a modern European transactional email service that prioritizes:

- **Simplicity**: Clean APIs and straightforward integration
- **Reliability**: High deliverability rates and robust infrastructure
- **Developer Experience**: Excellent documentation and developer-friendly tools
- **European Hosting**: GDPR-compliant infrastructure hosted in Europe
