Send emails to multiple recipients using arrays for the `to`, `cc`, and `bcc` fields.

## Multiple recipients

You can specify multiple recipients by passing an array of email addresses:

```typescript
import { sendEmail } from '#imports'

export default defineEventHandler(async () => {
  const result = await sendEmail({
    from: 'nuxt@lettermint.dev',
    to: ['ok@testing.lettermint.co', 'softbounce@testing.lettermint.co'],
    subject: 'Test Multiple Recipients',
    text: 'Testing multiple recipients'
  })

  return { success: true, result }
})
```

## CC and BCC recipients

Include CC (carbon copy) and BCC (blind carbon copy) recipients:

```typescript
import { sendEmail } from '#imports'

export default defineEventHandler(async () => {
  const result = await sendEmail({
    from: 'nuxt@lettermint.dev',
    to: ['ok@testing.lettermint.co', 'softbounce@testing.lettermint.co'],
    cc: ['ok@testing.lettermint.co', 'softbounce@testing.lettermint.co'],
    bcc: 'ok@testing.lettermint.co',
    subject: 'Test Multiple Recipients',
    text: 'Testing multiple recipients'
  })

  return { success: true, result }
})
```

<Info>
  You can use either a single email string or an array of email strings for `to`, `cc`, and `bcc` fields.
</Info>

## Reply-To configuration

Set custom reply-to addresses for your emails:

```typescript
import { sendEmail } from '#imports'

export default defineEventHandler(async () => {
  const result = await sendEmail({
    from: 'nuxt@lettermint.dev',
    to: 'ok@testing.lettermint.co',
    replyTo: 'support@example.com',
    subject: 'Test Email',
    html: '<h1>Hello World</h1>'
  })

  return { success: true, result }
})
```

## Named email addresses

Use objects with `email` and `name` fields for more descriptive recipients:

<Tabs>
  <Tab title="Single recipient">
    ```typescript
    import { sendEmail } from '#imports'

    export default defineEventHandler(async () => {
      const result = await sendEmail({
        from: {
          email: 'nuxt@lettermint.dev',
          name: 'Nuxt Team'
        },
        to: {
          email: 'ok@testing.lettermint.co',
          name: 'Test User'
        },
        subject: 'Welcome!',
        html: '<h1>Welcome to our service</h1>'
      })

      return { success: true, result }
    })
    ```
  </Tab>

  <Tab title="Multiple recipients">
    ```typescript
    import { sendEmail } from '#imports'

    export default defineEventHandler(async () => {
      const result = await sendEmail({
        from: {

        html: '<h1>Important team announcement</h1>'
    ```

  </Tab>

## Type definition

The email address types support both string and object formats:

```typescript
interface LettermintEmailAddress {
  email: string
  name?: string
}

interface LettermintEmailOptions {
  from: string | LettermintEmailAddress
  to: string | string[] | LettermintEmailAddress | LettermintEmailAddress[]
  cc?: string | string[] | LettermintEmailAddress | LettermintEmailAddress[]
  bcc?: string | string[] | LettermintEmailAddress | LettermintEmailAddress[]
  replyTo?: string | string[] | LettermintEmailAddress | LettermintEmailAddress[]
  // ... other fields
}
```

<Note>
  When using named addresses, the name will appear in the recipient's email client (e.g., "Nuxt Team \<nuxt@lettermint.dev\>").
</Note>
